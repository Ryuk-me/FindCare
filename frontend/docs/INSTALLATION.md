#

## 1. Installation

- `npm install --global yarn` (INSTALL YARN IF NOT INSTALLED)

- `yarn install` (don't use npm) (use this command in frontend root directory)

## 2. Run SvelteKIT

- `yarn dev`

## 3. Adding a package

- `yarn add <package-name>`

## 4. Removing a package

- `yarn remove <package-name>`
